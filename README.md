# Weather App 🌤️

  A full-stack weather application built with React and Express.js. Search any location worldwide, view a 12-day forecast, and save your
  favorite locations.

## 🌐 Live Demo
https://weather-delta-tawny.vercel.app

  ## ✨ Features

  - Search any location worldwide with autocomplete
  - Current weather conditions and 12-day forecast
  - GPS-based location detection
  - Save and manage favorite locations
  - Temperature toggle between Celsius and Fahrenheit
  - User accounts with email verification and password reset
  - Admin panel for managing users

  ## 🛠️ Tech Stack

  **Frontend**
  - React
  - Vite
  - HTML + CSS

  **Backend**
  - Node.js + Express.js
  - Turso (SQLite)
